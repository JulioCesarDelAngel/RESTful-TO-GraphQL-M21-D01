# RESTful-TO-GraphQL-M21-D01

## Descripción de la aplicación
Aplicación MERN stack que utiliza el motor de búsqueda de API de Google Books.
Refactorizada de API RESTful a API GraphQL con Apollo Server y JWT.

## Tecnologías utilizadas

### Servidor
* apollo-server-express
* bcrypt
* express
* graphql
* jsonwebtoken
* mongoose

### Cliente
* @apollo/client
* @testing-library/jest-dom
* @testing-library/react
* @testing-library/user-event
* bootstrap
* graphql
* jwt-decode
* react
* react-bootstrap
* react-dom
* react-router-dom
* react-scripts
